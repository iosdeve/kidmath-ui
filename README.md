## 项目介绍

如果你们家有上幼儿园的小朋友，如果你也正在教孩子学习算术，我想你一定遇到下面的问题

> 在辅导孩子学习算术的时候，为了避免小朋友忘记前面的学习，要经常的出一些算术题目给孩子做，不管是写在纸上的试题形式或者口头提问的形式

为了节省时间同时也为了帮助小朋友快速的学习算术知识，增加学习的趣味性，所以就开发了这个算术学习小工具
